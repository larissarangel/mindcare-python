# MindCare 🧠💙

Aplicativo desenvolvido em Python com o objetivo de auxiliar no acompanhamento da saúde mental de forma simples e acessível.

## 📌 Funcionalidades
- Registro diário de emoções
- Visualização do histórico emocional
- Dicas de bem-estar

## 🎯 Objetivo
Promover a saúde mental e a inclusão digital por meio da tecnologia.

## 💻 Tecnologias utilizadas
- Python
